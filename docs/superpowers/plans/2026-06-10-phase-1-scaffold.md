# FlowGate Phase 1: Scaffold + GitHub Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Working monorepo skeleton — NestJS backend, Angular frontend, PostgreSQL via docker-compose — pushed to a public GitHub repo, with both quality gates (`tsc --noEmit` + lint) green.

**Architecture:** Monorepo with `backend/` (NestJS, later DDD-layered) and `frontend/` (Angular standalone) as independent npm projects; PostgreSQL runs in Docker for dev. No app code yet — Phase 1 only proves the toolchain end to end.

**Tech Stack:** Node 24 · NestJS (latest, strict TS) · Angular (latest, standalone, SCSS, no SSR) · PostgreSQL 17 (Docker) · GitHub via `gh` CLI.

**Out of scope (later phases):** Auth (Phase 2), Requests CRUD (Phase 3), state machine/review (Phase 4), WebSocket (Phase 5), deploy/case study (Phase 6). One plan per phase, written when the phase starts.

### Task 3: docker-compose with PostgreSQL

**Files:**
- Create: `docker-compose.yml`
- Create: `.env.example`

- [ ] **Step 1: Write `docker-compose.yml`**

```yaml
services:
  postgres:
    image: postgres:17-alpine
    container_name: flowgate-postgres
    environment:
      POSTGRES_USER: ${POSTGRES_USER:-flowgate}
      POSTGRES_PASSWORD: ${POSTGRES_PASSWORD:-flowgate_dev}
      POSTGRES_DB: ${POSTGRES_DB:-flowgate}
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U flowgate -d flowgate"]
      interval: 5s
      timeout: 3s
      retries: 5

volumes:
  postgres_data:
```

- [ ] **Step 2: Write `.env.example`** (committed; real `.env` stays gitignored)

```
# PostgreSQL (dev defaults — match docker-compose.yml)
POSTGRES_USER=flowgate
POSTGRES_PASSWORD=flowgate_dev
POSTGRES_DB=flowgate
DATABASE_URL=postgres://flowgate:flowgate_dev@localhost:5432/flowgate
```

- [ ] **Step 3: Start and verify health**

Run: `docker compose up -d && sleep 8 && docker compose ps`
Expected: `flowgate-postgres` with status `running (healthy)`.

Run: `docker compose exec postgres psql -U flowgate -d flowgate -c 'select 1;'`
Expected: prints `1`.
